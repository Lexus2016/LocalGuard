# LocalGuard

> Protect your secrets from AI agents

**LocalGuard** — локальний проксі для маскування конфіденційних даних перед відправкою до LLM-провайдерів. Працює з будь-якими AI-агентами: Claude Code, ChatGPT, Codex CLI, Aider та іншими.

## Що робить

- **Перехоплює** API-ключі, токени, паролі, PII, криптогаманці, PEM-ключі
- **Працює 100% локально** — жодних даних не залишає вашої машини
- **Додає <50 мс затримки** на запит
- **Підтримує** macOS, Linux, Windows

## Тарифи

| Тариф | Ціна | Що входить |

## Завантаження

- macOS (.dmg) · Linux (.AppImage / .deb) · Windows (.exe)
- Homebrew: `brew install --cask localguard`
- CLI binary — [релізи на GitHub](https://github.com/Lexus2016/LocalGuard/releases)

## Швидкий старт

1. Встановіть LocalGuard
2. Активуйте ліцензію
3. Оберіть режим detection: Full / Regex Only / Off
4. Налаштуйте своїх AI-агентів (Claude Code, Codex CLI тощо)
5. Слідкуйте за Dashboard та Logs
